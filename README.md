# Tequila Lounge

Latin food, cocktails, tequila, and private events in Gastown, Vancouver.

## Tech Stack

- Next.js 14 App Router
- React 18
- Tailwind CSS
- Lucide React icons

## Getting Started

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Useful Commands

```bash
npm run lint
npm run typecheck
npm run build
```
